The <abbr title="Minimal Viable Product">MVP</abbr> of the Multitenant Analytics Application (MtAA) will be a dashboard for a law firm to track case metrics

## Core Features

### User Facing Dashboard

- Data visualization using web components (real-time?)
- Customizable components with a drag and drop interface

### Internal Admin Tools

- User management
- Permissions and role-based access control
- A/B testing configuration interface
- Analytics of dashboard usage

## Technology

- Nest.js
- TypeORM
- PostgreSQL
- Redis
- Docker
- GraphQL
- WebSocket
- Astro
- Webcomponents
